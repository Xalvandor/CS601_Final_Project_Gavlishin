CS Projects

Here are a few projects I have worked on during my time at BU MET's MS in Software Development program. For more details, please visit my GitHub.

CS 521: House Plant Manager

I created an application using Python to help care for house plants. With this program, you can save your houseplants, when you acquired them, and log when you water them. You will receive an alert if you are watering too often or not enough.

CS 622: Gig Management

This is an application built for musicians for managing their events and gigs. Musicians can add gigs and rehearsals to their calendar along with set lists and other event information, such as time, place, and fee. This application was built using Java and SQLite.

CS 669: Yoga Database

For this course, I designed and implemented a database for a yoga flow creation database. This was designed from the ground up, starting with use cases, then writing structural database rules, followed by ERDs, then writing the SQL with SQLServer to deploy the database on Azure. I then enhanced the database by adding indexes, stored procedures and queries, triggers, and history tables.

CS 601: Portfolio Website

Using html, css, and Javascript, I designed a responsive and dynamic website for showcasing my projects and career highlights.
